# Privacy Policy

ChatGPT View Manager is a local Chrome extension for managing long ChatGPT conversations.

This extension is not affiliated with OpenAI.

## Data Access

The extension runs on `https://chat.openai.com/*`.

It can read and modify the ChatGPT webpage DOM in order to provide features such as:

- detecting conversation blocks;
- adding the View Manager panel;
- showing hover controls;
- bookmarking conversation turns;
- collapsing/restoring conversation blocks;
- adding notes;
- setting text in the ChatGPT prompt composer through the Custom Prompt Dialog.

The extension does not intentionally collect or transmit ChatGPT conversation content.

## Local Storage

The extension stores data locally using Chrome extension storage.

Stored data may include:

- bookmark titles;
- bookmark notes;
- collapsed block records;
- block notes;
- conversation/page identifiers;
- UI settings such as theme, panel state, and dialog size/position.

This data remains on your device unless you export it or otherwise share your browser profile data.

## Custom Prompt Dialog

The Custom Prompt Dialog can set text in the ChatGPT prompt composer.

The dialog does not read or copy existing text from ChatGPT’s prompt composer into the extension.

Draft text in the Custom Prompt Dialog is kept only for the current tab/page lifetime and is not persisted across browser sessions.

## Import and Export

The extension allows View Manager data to be exported as a JSON file.

Exported files may contain user-written bookmark titles and notes. Treat exported files as personal data if you include sensitive information.

Imported files are processed locally by the extension.

## Network Access

ChatGPT View Manager does not send data to any external server controlled by this project.

The extension only runs on ChatGPT pages and interacts with the webpage already loaded in your browser.

## Deleting Data

You can remove stored extension data by:

1. Removing the extension from Chrome; or
2. Clearing the extension’s site/storage data from Chrome; or
3. Using Chrome extension storage clearing tools.

If import/export data has been saved to files, delete those files separately.

## Third-Party Services

ChatGPT is operated by OpenAI.

This extension modifies the local ChatGPT webpage experience in your browser, but it does not control how ChatGPT or OpenAI process data you enter into ChatGPT.

## Changes

This privacy policy may be updated as the extension changes.
